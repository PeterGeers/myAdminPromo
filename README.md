# myAdmin — Promotional Website

Marketing website for [myAdmin](https://myadmin.jabaki.nl), an integrated platform for STR (Short-Term Rental) property managers combining rental analytics with financial administration.

## Tech Stack

- **Framework**: Next.js (SSR/SSG)
- **Styling**: Tailwind CSS
- **Hosting**: AWS Amplify
- **Infrastructure**: Terraform

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Infrastructure is managed via Terraform in `infrastructure/amplify/`. See the [terraform.tfvars.example](infrastructure/amplify/terraform.tfvars.example) for required variables.
